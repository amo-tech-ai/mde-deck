# MDE.AI Pitch Deck Builder: Project Plan & Progress Tracker

## 1. Overview: What We Are Building

*   **Purpose:** To create an AI-powered tool that helps startup founders quickly build beautiful, professional, and effective pitch decks.
*   **Goal:** We want to empower anyone with a great idea to create a compelling story for investors, without needing design skills or spending weeks on formatting.
*   **Success Looks Like:** A user can go from a simple idea to a downloadable, investor-ready pitch deck in under 15 minutes.

## 2. Our Game Plan: Project Phases

Our project is broken down into five clear phases:

*   **Phase 1: Planning & Strategy:** Laying the foundation for success.
*   **Phase 2: Design & Prototyping:** Creating the visual look and feel.
*   **Phase 3: Building & Development:** Bringing the designs to life with code.
*   **Phase 4: Testing & Feedback:** Making sure everything works perfectly.
*   **Phase 5: Launch & Growth:** Sharing our creation with the world.

## 3. Tasks by Page: What We'll Create

#### Landing Page (The Welcome Mat)
- Write compelling headlines that grab attention.
- Design an engaging visual that shows off the final product.
- Create a clear "How It Works" section.
- Add testimonials from early supporters.
- Ensure the "Sign Up" button is easy to find and use.

#### Wizard (The Magic Interview)
- Write friendly, conversational questions for each slide (Problem, Solution, etc.).
- Design a simple, step-by-step progress bar.
- Create a clean input area for founders to type their ideas.
- Build the "AI, make this better!" feature to help refine text.
- Design an animated loading screen while the AI generates the deck.

#### Dashboard (The Command Center)
- Design a clean layout to display all of the user's pitch decks.
- Create a prominent "Create New Deck" button.
- Build simple options to rename, duplicate, or delete a deck.

#### Editor (The Creative Studio)
- Design an intuitive three-part layout: slides on the left, main view in the center, and editing tools on the right.
- Allow users to easily edit text and replace images.
- Build a simple "Brand" tool to apply company colors and a logo.
- Integrate an AI tool to generate new images based on text prompts.

#### Preview Mode (The Big Stage)
- Create a clean, full-screen presentation view.
- Allow navigation using arrow keys.
- Build a feature to generate a shareable link.

## 5. The Team: Roles & Responsibilities

*   **Product Designer:** Responsible for the overall look, feel, and user experience. Creates the wireframes and final visual designs.
*   **Frontend Developer:** Writes the code to build the user interface and connect it to our systems.
*   **Backend & AI Developer:** Connects the app to our database (Supabase) and the AI (Gemini). Makes sure data flows smoothly.
*   **Quality Assurance (QA) Tester:** Tries to "break" the app to find bugs and ensures everything works as expected on different devices.

## 7. Success Criteria: How We'll Know We've Won

We will consider this project a success when we meet these goals:

*   **Performance:** The app loads quickly, and the AI generates a deck in under 15 seconds.
*   **Usability:** A new user can create their first deck without needing a tutorial.
*   **Quality:** The final exported deck looks professional and is ready for an investor meeting.
*   **Responsiveness:** The entire experience works beautifully on desktops, tablets, and mobile phones.
*   **Reliability:** The app is stable, with minimal bugs or crashes.
